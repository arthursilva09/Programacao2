1 - Baixar e instalar python
2 - Baixa e instalar pip
3 - Criar virtualenv 
	Windows: py -m venv myproject
	Linux/MacOS python -m venv myproject
4 - Executar virtualenv
	Windows: myproject\Scripts\activate.bat
	Linux/MacOS: source myproject/bin/activate
5 - Instalar o django
	Windows: py -m pip install Django
	Linux/MacOS: python -m pip install Django
6 - Criar projeto
	django-admin startproject programacao2
7 - Run
	py manage.py runserver
8 - Acesso
	127.0.0.1:8000 ou localhost:8000
